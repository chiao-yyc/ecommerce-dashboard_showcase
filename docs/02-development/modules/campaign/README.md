# Campaign 行銷活動管理系統

## 系統概述

Campaign 管理系統是專為電商平台設計的完整行銷活動管理解決方案，支援活動創建、管理、效果追蹤與歸因分析。

## 核心功能

### 活動管理
- 活動創建、編輯、啟用/停用
- 活動狀態管理與生命週期追蹤
- 批量操作支援

### 效果分析
- 多維度歸因分析
- 重疊分析與競爭強度評估
- 趋勢分析與績效追蹤

### 配置管理
- 靈活的活動類型系統
- 自定義屬性與規則配置
- 假期數據同步機制
